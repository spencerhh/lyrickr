lyrickr

about

I've created this website to combine two of my passions: web development and music. This site uses multiple different APIs to retrieve data about a specified music artist and a (optional) song of theirs. Currently, the site uses lyrics.ovh and TasteDive to retrieve data and I plan on slowly implementing more APIs as I have more time and gain more access.

search

To use this site, simply type in an artist's name and a (optional) song they've made. This API is case-insensitive, but the spelling must be exact. Unfortunately, this API doesn't allow for the artist to be featured on the song when searching.
